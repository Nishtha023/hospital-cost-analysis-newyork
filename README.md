# Hospital-cost-analysis-newyork
Big data analysis of 1M+ hospital discharge records from New York State (2009-2017) examining cost drivers and patterns in healthcare. Uses MongoDB Compass for data cleaning and Tableau for visualization. Key findings: 0.98 charge-cost correlation, severity-based pricing validation, and actionable insights for hospitals, policymakers, and patients.
